# Servicios y procesos en Linux

## Objetivo

Aprender a monitorear procesos y servicios del sistema operativo.

## Comandos utilizados

### Ver estado de un servicio

```bash
systemctl status ssh
```

### Ver todos los procesos

```bash
ps -ef
```

### Ver procesos en tiempo real

```bash
top
```

### Ver servicios en ejecución

```bash
systemctl list-units --type=service --state=running
```

### Ver tiempo de actividad del sistema

```bash
uptime
```

## Resultado

Se verificó que el servicio SSH se encuentra activo y funcionando correctamente.

También se practicó el monitoreo básico de procesos y servicios mediante herramientas administrativas de Linux.
