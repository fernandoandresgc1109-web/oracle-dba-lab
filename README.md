# Oracle DBA Lab

Laboratorio personal para aprender administración de bases de datos Oracle sobre Linux.

## Objetivo

Prepararme para un cargo de DBA Junior mediante prácticas reales con:

- Linux
- Oracle Database
- Usuarios y permisos
- Servicios
- Backups
- Recuperación
- Monitoreo

## Entorno de laboratorio

- VirtualBox
- Ubuntu Server 26.04 LTS
- RAM: 8 GB
- CPU: 2 núcleos
- Disco: 80 GB
- Red: Adaptador puente
IP actual: 192.168.1.178
Acceso remoto: MobaXterm por SSH
