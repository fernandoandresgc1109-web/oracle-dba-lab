# Administración Linux para DBA

## Gestión de usuarios

### Ver usuario actual

```bash
whoami
```

### Ver información detallada

```bash
id
```

### Ver grupos

```bash
groups
```

## Gestión de procesos

### Ver procesos

```bash
ps -ef
```

### Monitor en tiempo real

```bash
top
```

## Gestión de servicios

### Estado SSH

```bash
systemctl status ssh
```

### Servicios activos

```bash
systemctl list-units --type=service --state=running
```
